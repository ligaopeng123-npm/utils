### 使用方式

```tsx
rtc create <template> // 创建项目

rtc c <template> 
    
rtc createModule <module> // 创建模块

rtc cm <module> 

rtc list  // 查看命令支持

rtc ls
```

### 插件依赖

| 插件              | 目的         |
| ----------------- | ------------ |
| chalk             | 美化输入输出 |
| prompts           | 用户信息输入 |
| download-git-repo | 远程模板下载 |
| cac               | cli指令      |
| lodash.template   | 处理模板     |
| ora               | loading效果  |
| fast-glob         | 文件名读取   |
